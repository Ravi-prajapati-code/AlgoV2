# 22 — Final Recommendation

**Date**: 2026-07-07. Answers the five mandated questions, nothing else. Evidence base: docs
14–21. Confidence labels per the project convention.

## 1. Is the signal genuinely valuable?

**Yes — within this universe and window. [High confidence on sign, Medium on magnitude.]**
The permutation test is the strongest single piece of evidence this project has produced: the
real RS ranking beat all 40 information-destroyed permutations of itself on both CAGR and Sharpe
(p=0.024), worth ~+9.6pp/yr on deployed capital under identical timing, filters, and costs.
Caveats that cap the magnitude: the universe is contaminated (docs/14), the parameters saw this
window (docs/16.6 Issue 2), and sub-period stability is untested (docs/21 item 2). None of these
plausibly flips the sign; any of them could halve the magnitude.

## 2. Is portfolio construction the bottleneck?

**Yes — and now it is measured, not inferred. [High confidence.]**
The wrapper costs ~14.3pp/yr (equal-weight universe +17.2% vs +2.9% for the random-ranking arm);
the signal buys back ~9.6pp; net -4.9pp vs passively holding its own universe. The audit
(docs/18) locates the recoverable part in three places: **stranded capital** (~27% of equity idle
at 0% even with slots full — the conviction-sizing buckets never redeploy their haircuts),
**friction** (3.2pp/yr at 8.46x turnover, majority generated by a sub-31-day cohort that loses
money before costs), and possibly **entry timing at slot-open** (suggestive, t≈1.7). Critically,
the sell side was measured and cleared — exits, stops, and crash-protection all test clean. The
bottleneck is deployment, not signal and not exits.

## 3. Can the current signal realistically beat the benchmark?

**On raw CAGR against the 2022-26 Midcap 150 (+21.05%): unlikely. [Medium-High confidence.]**
The arithmetic ceiling: own-universe passive is +17.2%; recovering the full measured friction and
stranded-capital leaks bounds a perfect wrapper near +15–17% — still below +21%, because ~3.9pp
of the gap is universe composition, which is out of scope by mission constraint. Beating a
top-quartile midcap bull period on CAGR was never what this construction (beta ~0.5, 73%
exposure) could do.

**On risk-adjusted terms (Sharpe/Calmar) against a normal midcap regime: genuinely open.
[Low-Medium confidence either way.]** Baseline Sharpe 0.59 vs benchmark 0.85 (Midcap window). If
E1+E2 recover 2–4pp of CAGR at similar volatility, the gap closes to contention; the stopping
rule in docs/19 makes this falsifiable within a few experiment cycles. Honesty requires noting
the benchmark's 21% CAGR is itself a historically hot sample.

## 4. If yes — what evidence is still missing?

In dependency order:

2. **E1 + E2 gate results** — the stranded-capital and cash-parking recoveries surviving the full

3. **Decontaminated universe re-run** (point-in-time watchlist) — external validity for every
   number above.
4. **Live reconciliation** (charges model + implementation shortfall) — the friction leak
   confirmed against real contract notes.
5. **Forward shadow track record** — the only evidence that can no longer be manufactured by
   this research process. Start immediately; nothing substitutes for it.

## 5. If no — what is the next major research direction?

If the stopping rule fires (E1+E2 fail to bring TEST Sharpe to benchmark contention), the honest
conclusion is that this signal, in this universe, at this concentration, is not a standalone
benchmark-beating product — and the correct next direction is the one docs/16.5 already
anticipated: **deploy the proven selection signal as a component, not a product.** Concretely:
a low-beta (~0.5), positive-Jensen's-alpha (+2.7%/yr point estimate) return stream with 42-54%
down-capture is a portfolio *diversifier* whose value should be tested with a mean-variance
spanning test against a passive midcap core (docs/16.6 Issue 5 experiment) — i.e., "does adding
this stream to an ETF core improve the combined frontier?" — rather than by asking it to outrun
the ETF alone. That reframing preserves everything proven (the signal, the clean exit stack) and
discards only the claim the evidence never supported.
